ScrollTo

Github Repo

Setup

Remember to include the plugin AFTER jQuery

Simple Usage


$("#scrollButton").click(function(){
	
	// the first parameter is the target, 
	// the second is the duration.
	$.scrollTo( $("#scrollTarget"), 400 );

});

Other Targets


// absolute number
$.scrollTo( 150, 400 );

// relative number (incrementing)
$.scrollTo( '+=100px', 400 );

// absolute number (object)
$.scrollTo( { top: '200px', left: '50px'}, 400 );

Waypoints

Github!

Basic Waypoint

Seems complex, until you read the documentation in depth.



// the method
waypoint(function);

//or...
waypoint(function, options);

// or just...
waypoint(options)

Break it down

The first one


$(".place-to-trigger").waypoint(function(){
	// do stuff
});

The second one


// basic
$(".place-to-trigger").waypoint( function(){}, {} );

// formatted
$(".place-to-trigger").waypoint(function(){
	// do stuff
}, {
	offset: '25%' // trigger when 25% up the page
});

Options

The third one


$(".place-to-trigger").waypoint({
	handler: function(){
		// do stuff
	},
	offset: '25%'
});

Another way to do the third one


function thingToDoOnWaypoint() {
	// stuff!
};

$(".place-to-trigger").waypoint({
	handler: thingToDoOnWaypoint,
	offset: '25%'
});
